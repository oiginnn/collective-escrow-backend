# Product MVP v1

## Roles
- User (buyer/supporter)
- Creator (invited only)
- Admin

## Screens
1) Home (feed)
- List of lots (image, creator, progress bar, time left)
- Actions: Participate / Buy now / Support

2) Lot page
- Photos, description
- Progress bar (no target number shown)
- Time left
- Delivery included in goal (USDT)
- Actions: Participate (1 per user), Buy now, Support

3) Wallet
- Balance
- History: top-up, participation, refund, support, payout

4) Profile
- Country
- My participations / wins / supports
- Support / disputes

5) Creator кабинет (invited)
- Create lot (draft → moderation → active)
- Upload shipping proof
