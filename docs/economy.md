# Economy MVP v1 (USDT)

## Units
- Participation price is fixed per lot (set by creator)
- 1 participation per Telegram user per lot
- Support = donation (any amount)

## Lot lifecycle
- active → funded → shipping → delivered → completed
- active → expired → refunds

## Money rules
- Participate: funds reserved (escrow) until delivery confirmation
- Buy now: closes lot immediately, refunds all participants automatically
- Expired: automatic refunds to all participants
- Platform fee: TBD (can be 0% at first); can apply only to support or to successful lot

## Delivery
- Delivery cost included in lot goal
